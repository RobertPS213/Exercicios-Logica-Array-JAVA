<h2>Exercícios de Lógica: Arrays em Java</h2>

Este repositório reúne uma série de exercícios práticos focados no domínio de lógica de programação utilizando a linguagem Java. O objetivo principal aqui foi exercitar a manipulação de estruturas de dados do tipo Array (vetores), controle de fluxo e algoritmos de busca e ordenação simples.

<h2>O que foi desenvolvido?</h2>
Os exercícios abordam situações comuns de lógica, como:

Manipulação de Vetores: Criação, inserção e leitura de dados em arrays.

Cálculos Matemáticos: Soma, média e busca de valores máximos e mínimos dentro de conjuntos de dados.

Processamento de Dados: Filtros de informações baseados em condições específicas dentro de laços de repetição (for, while).

<h2>🛠️ Tecnologias Utilizadas</h2>

☕ Java: Linguagem utilizada para a implementação de toda a lógica.

💻 Estrutura de Dados: Foco exclusivo em Arrays (Vetores).

<h2>📂 Como testar os exercícios</h2>

Certifique-se de ter o JDK instalado em sua máquina.

Clone o repositório: git clone https://github.com/RobertPS213/Exercicios-Logica-Array-JAVA.git.

Importe os arquivos .java em sua IDE de preferência (IntelliJ, Eclipse ou VS Code).

Execute as classes individualmente para ver a saída no console.
